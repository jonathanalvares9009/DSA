## For a program to run efficiently, both of these should be the least:

1.  Time Complexity

    Time complexity of an algorithm quantifies the amount of time taken by a rpogram to run as a function of length of the input.

2.  Space Complexity

    The space complexity of an algorithm or a computer program is the amount of memory space required to solve an instance of the computational problem as a function of characteristics of the input. It is the memory required by an algorithm to execute a program and produce output.

Can be measured in three ways: **Worst Case [O(big oh) notation]**, **Best Case [Ω(nig omega) notation]** and **Average Case [Θ(big theta) notation]**.
